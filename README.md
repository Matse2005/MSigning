# Inschrijven

Ik heb het maken en het uitbreiden van dit inschrijfplatform gebasserd op een gelijkaardig platform voor het inschrijven van studenten voor het [**KSDiest**](https://www.ksdiest.be/) bij activiteiten voor de [**sportdag**](https://sportdagksd.be).
Bij verder nadenken over dit platform ben ik tot de conclusie gekomen dat de basis tot zoveel meer uitgebreid kan worden dan het nu is.
Daarom ga ik het platform uitbreiden met de functionaliteiten die te vinden zijn in het bestand **txt/features.txt**.
